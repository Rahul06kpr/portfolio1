01 ——

Industrial Fault Detection Dashboard

A real-time Streamlit dashboard for industrial equipment fault detection using a CNN (RobustFaultCNN) built with PyTorch. Processes raw sensor signals, extracts windowed features, computes kurtosis-based fault probability, and renders 3D signal surface and machine stress visualizations with Plotly. Supports synthetic data generation, CSV upload, and manual signal entry.

RobustFaultCNN — Conv1d → MaxPool → AdaptiveAvgPool architecture

Edge mode toggle for fast inference

ML-Driven CVD Risk Analysis

A comprehensive cardiovascular disease early-detection app built with Streamlit. Takes 20+ clinical inputs — blood pressure, cholesterol, triglycerides, CRP, homocysteine, BMI, lifestyle factors — and runs a rule-based heuristic engine to generate a risk parameter table and personalized clinical insights.
